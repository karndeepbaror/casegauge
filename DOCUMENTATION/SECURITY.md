## Security Policy

`Case Gauge` Titan follows a privacy-first design.

**Data Handling:**
- No user data is stored
- No server-side processing
- All analysis runs locally in the browser

**Responsible Disclosure:**
If you discover a security issue or vulnerability:
- Do not exploit it
- Report it responsibly via GitHub Issues

**Privacy Commitment:**
This project does not collect, track, or transmit any `personal` or `investigative data.`
